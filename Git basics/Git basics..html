Имя пользователя Первое, что вам следует сделать после установки Git'а, — указать ваше имя и адрес электронной почты. Это важно, потому что каждый коммит в Git'е содержит эту информацию, и она включена в коммиты, передаваемые вами, и не может быть далее изменена: $ git config --global user.name "John Doe" $ git config --global user.email johndoe@example.com Выбор редактора $ git config --global core.editor emacs Утилита сравнения Другая полезная настройка, которая может понадобиться — встроенная diff-утилита, которая будет использоваться для разрешения конфликтов слияния. Например, если вы хотите использовать vimdiff: git config --global merge.tool vimdiff Проверка настроек git config --list Как получить помощь? Если вам нужна помощь при использовании Git'а, есть три способа открыть страницу руководства по любой команде Git'а: $ git help <команда> $ git <команда> --help $ man git-<команда>
